# 🎬 TMDB 5000 영화 데이터를 활용한 흥행 및 수익 요인 분석

## 📌 프로젝트 개요
본 프로젝트는 Kaggle의 **TMDB 5000 Movie Dataset**을 활용하여 영화의 흥행(수익)에 영향을 미치는 주요 변수들을 분석하고 시각화한 데이터 사이언스 과제입니다.

## 🛠️ 사용 도구
- **Language:** Python
- **Libraries:** Pandas, NumPy, Matplotlib, Seaborn
- **Editor:** Visual Studio Code

## 📊 주요 분석 내용
1. **제작 예산 vs 수익 관계:** 상관계수(0.73) 분석을 통해 예산 투입과 수익의 강한 양의 상관관계를 확인.
2. **장르별 가성비 분석:** 절대적 수익은 액션/SF가 높으나, 투자 대비 수익률(ROI)은 **'공포(Horror)'** 장르가 압도적임을 도출.
3. **영화 제목 길이의 영향:** 제목 길이에 따른 마케팅 효과와 평균 수익 간의 비선형적 관계 분석.
4. **연도별 트렌드 분석:** 시대 흐름에 따른 유행 장르(SF, 애니메이션 등)의 변화와 수익성 추이 분석.

## 💡 주요 인사이트 (Conclusion)
- 직관적으로 예상했던 변수들이 실제 데이터와는 다른 양상을 보이는 경우가 많음을 확인했습니다.
- 특히 가성비 측면에서는 저예산 고효율 장르(공포)의 전략적 가치가 높음을 발견했습니다.
- 데이터 기반 분석 전까지는 섣불리 흥행 요인을 단정 지을 수 없다는 '데이터 기반 의사결정'의 중요성을 깨달았습니다.
